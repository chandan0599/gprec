Bulk Update Due Fees (CSV Upload)

Tuition Fee and Hostel Fee amounts vary per student, so they're set here instead of in Base Fee Amounts above. Upload a CSV file (export it from Excel with File > Save As > CSV) to set each student's total fee and amount already paid, in bulk - the pending (due) amount shown on their dashboard is calculated automatically as Total Fee minus Paid Amount. Columns, in order: Roll Number, Fee Type, Detail, Total Fee, Paid Amount, Application (use semester-tuition for Tuition, room-rent or caution-deposit for Hostel). Select the due date that applies to this whole file below before uploading. Uploading replaces the pending fees list for every roll number included in the file. Rows where Paid Amount already covers the Total Fee are skipped (nothing left due).

i
  • The key saved here is used by the browser integration; for public deployment, route provider calls through a secured backend.
  • Avoid using a paid/production key in the browser because visitors can inspect network requests.
  • Google Gemini works directly from the browser; OpenAI and Anthropic usually require a server route.
  • If the AI request fails, GPRECian Bot automatically falls back to built-in keyword answers.
  • Role-specific real-data lookups answer first; live AI only handles general questions and should not be relied on for high-risk personal-record decisions.

Ollama runs on your own machine, so no API key is needed - just install Ollama, pull a model (e.g. ollama pull llama3.2), and make sure it's running (ollama serve or the Ollama app) at the URL above. Since browsers block cross-origin requests by default, Ollama also needs OLLAMA_ORIGINS set to allow this site's address (e.g. OLLAMA_ORIGINS=* for local testing) before the connection below will succeed. The visitor's browser calls it directly, so this only works while testing on the same machine Ollama is running on - it cannot serve AI answers to other visitors' browsers.

Google Calendar API (Indian Holidays)

i
  • Indian national holidays are fetched live from Google's public "Holidays in India" calendar.
  • No national holidays will show on the calendar until an API key is saved here.
  • Create a Google Cloud project, enable Google Calendar API, generate an API key, and restrict it to this site's HTTP referrer.
  • A referrer-restricted key is safe to store in the browser because it will not work from other sites.

Parent OTP SMS (MSG91)

Without this configured, the Parent Login OTP is generated and verified for real on the server but only ever shown on-screen in the browser (there's no SMS gateway by default). Saving valid MSG91 credentials here makes the OTP actually get texted to the parent's phone instead. You'll need an MSG91 account with a DLT-registered sender ID and an approved SMS template containing exactly one variable for the OTP - enter that variable's exact name below (MSG91 templates commonly use OTP or var1; it must match whatever you registered on the MSG91/DLT portal, or delivery will fail even with correct credentials).

Suggested DLT template text (register this exact wording on MSG91, with {{OTP}} as the one variable):
"Dear Parent, your OTP for GPREC Parent Portal login is {{OTP}}. It is valid for 5 minutes. Do not share this OTP with anyone, including GPREC staff. - G Pulla Reddy Engineering College"

WhatsApp fallback (optional)

If SMS isn't configured above, or a send fails, the OTP is delivered over WhatsApp instead (using the same MSG91 Auth Key) before falling back to on-screen display. Needs a WhatsApp Business number connected in MSG91 and an approved WhatsApp template with one body variable for the OTP.

Suggested WhatsApp template text (register this on MSG91/Meta, with {{1}} as the one body variable):
"Dear Parent, your OTP for GPREC Parent Portal login is {{1}}. It is valid for 5 minutes. Do not share this OTP with anyone, including GPREC staff. - G Pulla Reddy Engineering College"

Grades notification (Exam Cell)

Lets the Exam Cell send a one-off SMS/WhatsApp to a parent from the Grades Preview screen after looking up a student's result. Needs its own approved template (different wording than the OTP one above) with a single variable carrying the full result summary.

Suggested SMS template text (with {{MESSAGE}} as the one variable):
"Dear Parent, {{MESSAGE}}. For full details, login to the GPREC Parent Portal. - G Pulla Reddy Engineering College"

Suggested WhatsApp template text (with {{1}} as the one body variable):
"Dear Parent, {{1}}. For full details, login to the GPREC Parent Portal."

Vehicle Pass Verification (KYC)

Sarathi (driving license) and VAHAN (vehicle registration) don't offer a public API directly. API Setu, the Government of India's own API gateway (run by NeGD/MeitY), exposes official DL and RC verification APIs from the Ministry of Road Transport at no per-call cost - this card is built for that. It requires registering GPREC as a consumer organization on API Setu first (a one-time government approval process, not instant); once approved, paste the base URL/paths/key API Setu issues you below.

Without this configured, Vehicle Pass applications still work normally - they're just marked "Not configured, manual review required" so you can check the uploaded license/RC photos yourself before approving.
